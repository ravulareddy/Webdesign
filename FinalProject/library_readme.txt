/// Library and Help

Steps to run the application :

1. create database using the dump.sql ( select the entire content in dump sql and click on run)
2. open connect.php and change the user name and password to your mysql workbench
3. copy the library and help folders into the mamp folder 
4. start the server( apache and mysql)
5. open localhost url and append /library/library.html to access the page 
6. Sample user name and password to login into the application (test, test) when you click on borrow book
